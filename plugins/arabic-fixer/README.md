# Arabic Encoding Fixer Plugin

## الوصف
هذه الإضافة مخصصة لإصلاح مشكلة الرموز الغريبة (Mojibake) التي تظهر في النصوص العربية بعد ترقية MyAds من إصدار قديم (v3.x.x) إلى الإصدار الجديد (v4.x.x). تحدث هذه المشكلة عادةً عندما يتم استيراد قاعدة بيانات بترميز `latin1` كانت تحتوي على بيانات `UTF-8`.

## المميزات
- فحص تلقائي لجميع الجداول الحساسة للنصوص.
- إصلاح النصوص المعطوبة (مثل `Ø§Ù„Ø¹Ø±Ø¨ÙŠØ©`) وتحويلها إلى عربية صحيحة.
- استثناء تلقائي للأعمدة الحساسة مثل (كلمات المرور، البريد الإلكتروني، الروابط، العناوين الرقمية IP).
- واجهة تحكم بسيطة مع زر تشغيل وتأكيد.
- مدمجة في القائمة الجانبية للوحة الإدارة (Sidebar).

## الجداول المدعومة
تقوم الإضافة بفحص الجداول التالية:
`setting`, `options`, `ads`, `news`, `forum`, `f_coment`, `cat_dir`, `f_cat`, `directory`, `users`, `messages`, `report`, `banner`, `link`, `notif`, `visits`.

## طريقة الاستخدام
1. قم برفع مجلد الإضافة إلى مسار `plugins/arabic-fixer`.
2. من لوحة التحكم، اذهب إلى **المظهر > الإضافات**.
3. قم بتفعيل إضافة **Arabic Encoding Fixer**.
4. سيظهر رابط جديد في القائمة الجانبية باسم **إصلاح العربية (Mojibake)**.
5. اضغط على الرابط، ثم اضغط على زر **بدء عملية إصلاح اللغة العربية**.

> [!IMPORTANT]
> يرجى أخذ نسخة احتياطية (Backup) لقاعدة البيانات قبل تشغيل أداة الإصلاح.

## المتطلبات
- PHP 8.1 أو أحدث.
- مكتبة `mbstring` مفعلة في السيرفر.
- نظام MyAds الإصدار الرابع.
